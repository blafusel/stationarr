EPG Guide

The main view shows your Plex content as TV channels with a live broadcast schedule. Each row is a channel; each block is a program airing at that time. The current time is marked with a vertical line.

  • Click a program to start playback. If the program is currently airing, playback resumes from the live broadcast position.
  • Hover a program to see a tooltip with poster art, description, year, and rating.
  • Yellow-highlighted bars are currently airing.

Channels & Grouping

Channels are grouped into collapsible sections: Libraries, Video Playlists, Music Playlists, Categories, and Collections.

  • Click a section header to collapse or expand it.
  • Drag the ⠿ grip on any channel row to reorder it within its group. Order is saved across reloads.
  • Toggle grouping on/off in Settings → Display.

EPG Time Scale

  • Drag the ⟺ Scale handle (top-left of the time bar) left or right to zoom the EPG in or out.
  • Alternatively, use the EPG time scale slider in Settings → Display.
  • Scale is saved across reloads.

Search

Type in the search bar to filter channels and programs in real time. Press × or clear the field to reset.

Shuffle

Click ⇄ Shuffle in the header to instantly play a random item from any visible channel. Each subsequent item will also be chosen randomly until you turn shuffle off. The button is highlighted in yellow when active.

Video Player

  • ⏮ / ⏭ — skip to the previous or next program on the same channel.
  • ⇄ — play a random program from any channel.
  • ← / → arrow keys — previous / next when the player is open.
  • Playback position is saved automatically and restored on next play.
  • Minimise the player with the ⤓ button to keep watching while browsing the guide.

Music Player

Music playlists open a Plexamp-style player with blurred album art backdrop, seek bar, and volume control.

  • Space — play / pause
  • ← / → — previous / next track
  • Esc — close or minimise to mini-player

Stable Broadcast Schedule

When enabled (Settings → Playback), the schedule is deterministic — the same content always airs at the same time on every reload, just like a real TV channel. Disable it for a randomised schedule.

Settings

Click ⚙️ Settings to open the configuration panel. It is divided into sections:

  • Server — Plex URL, auth token, connection test.
  • Content — choose which libraries, playlists, categories, and collections appear as channels.
  • Display — EPG scale, grouping, posters, animations, tooltips.
  • Playback — stable schedule, broadcast-position resume, position memory, auto-play, volume.
  • Auto-Refresh — toggle and set interval for automatic content refresh.
  • Advanced — content caching, low bandwidth mode, debug logging.

Configuration File

Your Plex server URL and token can be pre-configured in config.json on the server — they will be loaded automatically on startup. See the README for setup instructions.
